## ğŸš€ Getting Started

### **Prerequisites**

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic knowledge of HTML, CSS, and JavaScript
- A code editor (VS Code, Sublime Text, etc.)

### **Installation**

```bash
# Clone the repository
git clone https://github.com/SaraSalah1/js-mini-projects.git

# Navigate to the project directory
cd js-mini-projects

# Open any project
cd Day01-QR-Code-Generator
open index.html
```

### **Running Locally**

Each project can be run by simply opening the `index.html` file in your browser. For better development experience, you can use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```
